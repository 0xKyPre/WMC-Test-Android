# Android Development

## Create a new Project

First start Android Studio and make a new Project. Depending on the Test-Assignment choose between an `Empty Activity` or `Navigation UI Activity`

![activity](./img/as_c.png)

Select `Android 10.0` and choose a name and safe folder.

![conf](./img/n_p.png)

## Coding

As soon you have opened your new project, build it. Missing imports or dependencies will be downloaded and all the errors will dissapear.
